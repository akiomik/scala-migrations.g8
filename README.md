scala-migrations.g8
===================

A [giter8](https://github.com/n8han/giter8) template for scala projects using [scala-migrations](https://github.com/blair/scala-migrations).

## Usage

  ```
g8 akiomik/scala-migrations
  ```

You can configure the following parameters.
  - name
  - datetime
  - organization
  - migration_name
